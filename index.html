    • Frequently Asked Questions

    • How does the dataset work for different versions of the Games?
    • All Dataset corresponds to the lastest versions of the games (including DLC).
    • How about different language versions of the games, like the Japanese Red/Green?
    • All Dataset only corresponds to the English releases of the games.
    • How does the dataset work for Pokemon Forms?
    • All of the data listed in the Pokemon Form is applied only to that specific Form, any missing Fields equals to the default Forms value.
    • How was all of the dataset imported?
    • Data from various sources was exported in to Google Sheets through scripts.
    • Can i use this dataset for a project of mine?
    • Sure, feel free to use it however you like.
    • I have suggestions/questions/would like to express my opinions, where do i contact you?
    • You can contact me directly here.
